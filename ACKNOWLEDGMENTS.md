# Acknowledgments

RetailVision AI builds on the excellent work of the following open-source projects:

- [Ultralytics YOLOv8](https://github.com/ultralytics/ultralytics) — object detection
- [OpenCV](https://opencv.org/) — computer vision and image processing
- [FastAPI](https://fastapi.tiangolo.com/) — backend API framework
- [Streamlit](https://streamlit.io/) — interactive dashboard
- [PostgreSQL](https://www.postgresql.org/) — data storage
- [psycopg2](https://www.psycopg.org/) — PostgreSQL adapter for Python

Thanks to the maintainers and communities behind these tools for making local,
open computer vision development accessible.
